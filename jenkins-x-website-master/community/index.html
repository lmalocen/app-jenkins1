Community

Please join us in the community, we’d love to hear your feedback!

Slack

The Kubernetes community hangs out on Slack at slack.k8s.io so we have rooms there to chat about all things Jenkins X:

  • #jenkins-x-user for users of Jenkins X
  • #jenkins-x-dev for developers of Jenkins X

IRC

Jenkins folks tend to use IRC so we have an IRC room #jenkins-x IRC room on freenode.net

You could try using the riot app for #jenkins-x

Issues

Got any thoughts, ideas or feedback then please drop them in a GitHub Issue

Email

If you prefer email then join us on the jenkins-developers mailing list. Prefix posts with [Jenkins X] in the subject.

Want to contribute?

It will be lots of fun!

Check out the contribution guide for some tips on how to get started
